About

My exciting journey down the digital rabbit hole

  • 2018-Now

    Full-Stack Training + Freelance Projects

    Last year, I turned the focus to development with a deep dive into full-stack training in a range of tech stacks and JS frameworks. I'm currently working several projects for independent clients and small businesses.

  • 2012-2018

    Full-Time Tutoring

    While dabbling in small brand design projects on the side, I was honored to spend six years tutoring students ages 6-22.

  • 2009-2015

    Digital Brand + Web Dev

    This window of time could also be titled "Startup Season"! I consulted for a handful of startups on business planning, design, and digital branding. I also wrote web content, edited, and tutored in my "spare time".

  • 2007-2009

    First Contact

    Ahh, where it all began...as Senior Editor of a digital zine, I got my first taste of HTML/CSS, CMSs, image editing, podcasts, SEO copywriting, newsletters, and more.

Wedding Site

A simple, single-page wedding site

This site is an in-progress template that will be filled in as the couple provides their wedding details. The goal was to create a single page app with a parallax scroll feature. The site will live at GitHub until it is finalized and as the couple decides on hosting.

Tech Stack: HTML | CSS | JavaScript

  • Date: 2019
  • Client: Dave + Gwen
  • Category: Code + Site Design

Mind Fuel

A simple tutoring + editing business site

This in-progress site is a personal project, designed to practice CSS positioning and mobile responsiveness. The goal was to gain practice altering the mobile view for a better user experience and to position items where desired.

Tech Stack: HTML | CSS | JavaScript

  • Date: March 2019
  • Client: Self
  • Category: Code + Site Design

Dream Fuel Designs

Former developer portfolio site

This was my first web dev portfolio site, designed and coded over 2 days. The feature I am most of proud of is the responsiveness across several devices, especially the Projects gallery and Connect form. I also immensely enjoyed creating the comic-book-like About page.

The creative challenge here was scalability - as the site grew, it became harder to manage, prompting a search for seamless, single page options.

Tech Stack: HTML | CSS | JavaScript

  • Date: December 2018
  • Client: Self
  • Category: Code + Site Design

Third Wave Coffee

A coffee educator + locator app

This project was designed as an experiment for a barista desiring to introduce people to this movement and help them find local third wave coffee shops. This was my first project in PHP/Laravel and it was a smooth ride. Login/authorization was the only significant challenge I faced on this one.

Tech Stack: PHP/Laravel | Valet | Postgresql | Nginx
Specs: Full CRUD app | MVC architecture | Login/Auth

  • Date: November 2018
  • Client: Barista (chooses to remain anonymous)
  • Category: Code + Site Design

upLIFT

A positivity app

This app was designed to help people capture and revisit the positive moments from their day. I would like to redesign this app with a focus on mobile responsiveness and features like image upload and an icon bank.

Tech Stack: MongoDB | EJS | Express | Node.js

Specs: Full CRUD | MVC architecture | Login/authorization

  • Date: October 2018
  • Client: Educational Project
  • Category: Code + Site Design

Clueless Closet

An outfit planning and closet organizer app

This was a collaborative project focused on helping people shop their own closet and style new looks. I was primarily responsible for the site design, including details like modals and icons. Our group chose to keep the design simple to provide a neutral palette for evaluating outfits.

Tech Stack: MongoDB | Express | AngularJS | Node.js
Specs: Full CRUD | MVC architecture | Login/authorization

  • Date: October 2018
  • Client: Educational Project - Collaborative
  • Category: Site Design + Some Code
